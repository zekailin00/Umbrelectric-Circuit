# Umbrelectric-Circuit


## Goal


- Two switches 
  - Power on or off
  - Moving umbrella up or down 
- Moving Up
  - Open the clamp - servo
  - Open the umbrella - stepper motor 
- Moving Down 
  - Close the umbrella 
  - Close the clamp
